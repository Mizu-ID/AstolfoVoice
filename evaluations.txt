● Ollie
Udah selesai, dan kompilasi udah gw verifikasi lulus (javac exit 0). Catatan penting dulu soal build: Gradle-nya gak bisa jalan karena disk sisa cuma 0.13 GB — pas mau extract gradle 9.6.1 gagal "not enough space". Jadi gw verifikasi pakai javac langsung pakai dependency dari gradle cache, dan semua source kompilasi bersih (cuma warning deprecated API bawaan paper-api, bukan error). Kalo lu butuh build fat-jar asli, lu harus bersihin disk dulu atau build di mesin lain.

Ini yang gw kerjakan:

Bug yang diperbaiki

1. Suara grup terdengar pelan/distorsi (ProximityResolver). Sebelumnya saat bypassforgroups=true, distance di-set ke voiceRange (48) → suara grup kena attenuation padahal harus jernih. Gw ganti ke distance besar finite (~1jt, attenuation ~0) biar group chat jernih kaya SVC asli, plus whisper flag grup selalu false.

2. Mute gak berfungsi sama sekali. AstolfoApiImpl punya field muted tapi gak pernah dicek di routing audio. Gw bikin MuteHolder (holder shared kaya AstolfoOverride), cek MuteHolder.isMuted(sender) di awal processMic → sender mute gak ngirim suara ke siapapun. API setMuted/isMuted sekarang sinkron ke holder, resetPlayer juga clear mute.

3. Playback "location" gak posisional (AstolfoCommand + AudioEngine + StreamingAudioPlayer). Sebelumnya /play <file> location ... cuma broadcast ke world tanpa koordinat. Gw tambah playAtLocation yang beneran kirim LocationSoundPacket (x,y,z) jadi suara muncul dari titik dunia, bukan di player.

4. kickPlayer rate-limit dari thread async (NetManager). player.kickPlayer() dipanggil langsung di handler packet yang bisa async → gak aman/bisa throw. Gw schedule ke main thread via compatibility.runTask().

5. /reload gak ber efek (AstolfoVoice + AstolfoCommand). Sebelumnya manggil plugin.reloadConfig() (Bukkit config) tapi field config (AstolfoConfig) gak diperbarui → perubahan jarak suara dll gak berlaku. Gw bikin reloadAstolfoConfig() yang beneran reload AstolfoConfig, dan command reload sekarang pakai itu + nunjukin nilai baru (range/bitrate/port).

6. Double-decrement activePlaybacks (AudioEngine.playQueue). Queue loop decrement counter padahal watcher per-part juga decrement → bisa negatif. Gw rapikan pakai reserveSlot() CAS + decrement cuma sekali per playback.

7. Bocor memori per-player saat quit (AstolfoVoice.onQuit). Override/mute/rate-limiter player gak di-clear. Gw tambah cleanup + hapus baris sampah limiter = null yang gak sengaja.

8. Path traversal di resolveFile (AudioEngine). Gw tambah cek ../:/absolut path + isInside() biar gak bisa baca file di luar folder audio.

9. Spectator interaction (ProximityResolver). Gw tambah gate spectator_interaction biar spectator gak dengar/bicara ke player biasa saat nonaktif.

Config yang diperluas (biar bebas edit)

Banyak key di config.yml yang gak punya getter → setengah mati. Gw tambah getter + validasi (clamp range, dll): sampleRate, frameSizeMs, audio.formats, audio.privatechannel., integration.
