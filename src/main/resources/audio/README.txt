AstolfoVoice - Folder Audio
===========================

Letakkan file audio di sini untuk dipakai perintah /astolfo play ...

Format yang didukung:
  - .mp3
  - .ogg
  - .wav

Contoh penggunaan:
  /astolfo play player agus musik.mp3
  /astolfo play world world boss_theme.ogg
  /astolfo play all announcement.wav
  /astolfo play location world 100 70 200 siren.mp3 48
  /astolfo play group raiders raid.mp3

Catatan:
  - File di-resample ke 48kHz mono internal (kualitas sesuai config audio.resample_quality).
  - Nama file case-insensitive, boleh tanpa ekstensi di command.
  - Hindari file terlalu besar untuk playback "all" (memori server).
  - Folder ini dibuat otomatis oleh plugin saat pertama kali jalan.
