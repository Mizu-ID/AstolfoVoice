## 2. Constraint Kompatibilitas SVC (PALING KRUSIAL — jangan rusak)

Client Simple Voice Chat biasa harus tetap connect **tanpa mod tambahan, tanpa
patcher**. Server AstolfoVoice bertindak persis seperti server SVC di sisi transport.
Hal yang WAJIB dijaga byte-exact:

- **Channel namespace:** `voicechat:*` (BUKAN `astolfo:*`). Client SVC mengirim ke
  channel ini; kalau salah namespace, handshake gak akan pernah jalan.
- **UDP envelope:** `[0xFF magic][UUID player(16B, MSB+LSB)][byte[] encryptedPayload]`.
  encryptedPayload = `Secret.encrypt([typeID byte][packet.toBytes])` = IV(12B) +
  AES-128-GCM ciphertext (tag 128-bit).
- **10 tipe paket** + field order (lihat `AstolfoVoice/docs/PROTOCOL_REFERENCE.md`):
  Mic(0x1), PlayerSound(0x2), GroupSound(0x3), LocationSound(0x4),
  Authenticate(0x5), AuthenticateAck(0x6), Ping(0x7), KeepAlive(0x8),
  ConnectionCheck(0x9), ConnectionCheckAck(0xA).
- **Audio:** Opus 48kHz, 20ms frame, 960 sample, mono, MAX_OPUS_PAYLOAD_SIZE=1275.
- **Handshake sequence:** request_secret (TCP plugin channel, bawa
  compatibilityVersion) → server balas secret (16-byte AES + port + UUID + codec +
  MTU + distance + keepalive + groups + voiceHost + recording) → client kirim
  AuthenticatePacket(uuid,secret) via UDP → server cek → unCheckedConnections +
  AuthenticateAck → client ConnectionCheck → connections + ConnectionCheckAck →
  broadcast PlayerStatesPacket (roster).

Semua bug fix & fitur server-side TIDAK boleh menyentuh hal di atas. Fix yang aman:
routing logic, cleanup, file resolve, config, thread-safety, DSP, UI. Fix yang
merusak: ubah channel name, ubah field order paket, ubah audio konstanta.

## 3. Tech Stack & Build

- **Java 25** (toolchain Gradle, `options.release = 25`).
- **Gradle 9.6.1** (wrapper disertakan di repo; install Gradle ada di
  `../gradle-9.6.1` sibling dir kalau perlu, tapi wrapper `gradlew.bat` cukup).
- **Paper API 1.21.8** (`compileOnly io.papermc.paper:paper-api:1.21.8-R0.1-SNAPSHOT`).
- **NMS-free:** pakai Paper API publik (`World#rayTraceBlocks`, Adventure
  `Component.translatable`, `Component` clickable). Tidak ada reflection-per-version
  seperti SVC asli. Trade-off: fitur penuh hanya Paper 1.21+; Spigot fallback fitur
  dasar (raytrace off).
- **Fat-jar:** `com.gradleup.shadow` 9.3.2. Relocate: `org.concentus` →
  `id.astolfo.libs.concentus`, `javazoom.jl` → jlayer, `com.jcraft.jogg/jorbis` →
  libs, `org.gagravarr` → libs.
- **Dependencies:** Concentus (Opus pure-Java, vendored jar `libs/concentus-1.0`),
  JLayer 1.0.1 (mp3), JOrbis 0.0.17 (ogg/vorbis), vorbis-java-core 0.8 (ogg demux,
  opsional), PlaceholderAPI 2.11.6 (compileOnly, soft-depend).
- **Build command:**
  `./gradlew :AstolfoVoice:shadowJar` → `AstolfoVoice/build/libs/AstolfoVoice-X.Y.Z.jar`
- Build pertama butuh online (tarik deps maven), setelah itu `--offline` cukup.
- **OS:** Windows (PowerShell 5.1 bawaan). CATATAN PENTING encoding di bawah.

### Catatan encoding PowerShell 5.1 (PENTING untuk AI berikutnya)

- PowerShell 5.1 default encoding BUKAN UTF-8. `Get-Content`/`Set-Content`/`Out-File`
  default Windows-1252 → **merusak em-dash `—` (E2 80 94) jadi mojibake `Ã¢`**.
- Saat edit file yang punya non-ASCII (terutama `docs/*.md`), WAJIB pakai:
  `$utf8 = [System.Text.UTF8Encoding]::new($false)` lalu
  `[System.IO.File]::ReadAllText($path, $utf8)` + `WriteAllText($path, $content, $utf8)`.
  JANGAN pakai `Get-Content`/`Set-Content` tanpa `-Encoding UTF8`.
- Jangan pakai emoji karakter non-ASCII di banner/footer yang ditulis via PS script
  (pakai ASCII-only di bagian yang di-prepend/append; em-dash isi asli biarkan utuh).
- `cmd` sering menelan `$_`/`$var` di inline `powershell -Command "..."` → selalu
  pakai script file (`-File script.ps1`) untuk operasi non-trivial.
- `git commit -m "..."` di cmd kadang bermasalah dengan quote → pakai
  `git commit -F .git/COMMIT_MSG.txt` (tulis pesan ke file dulu).
- Tool `run_command` punya timeout 120s; Gradle `--rerun-tasks` shadowJar kadang
  timeout di batas. Kalau exit null tapi build_log bilang SUCCESSFUL, itu OK (build
  selesai tepat di batas timeout tool, bukan error).

## 6. Fitur Lengkap (status v0.2.3)

### Kompatibilitas SVC
- Handshake + 10 paket byte-exact, roster client broadcast, compatibilityVersion
  per koneksi (default 20).

### Sound physics (server-side raytrace Paper API)
- **Difraksi:** suara bungkung di tepi obstacle. Tiang 1-block tetap kedengar;
  dinding 2-wide mendam. Gain `1/(1+extra*k)`.
- **Transmisi:** tembus dinding tipis, redaman material-dependent (wool/leaves
  menyerap besar, glass sedang, stone/obsidian opaque). Beer-Lambert `e^(-abs*thick)`.
- **Medium (air):** lowpass dalam + absorption high-freq eksponensial per meter
  `700*e^(-m*0.35)`. Makin jauh di air makin mendam.
- **Reverb:** density block solid + cek atap → ruang tertutup (bergema).
- **Tier 1** (default, murah): modulasi distance + whisper flag.
- **Tier 2** (opt-in per world): decode Opus → DSP bake (low/high/band-pass biquad +
  reverb FDN + noise gate + softClip) → re-encode. Efek "bergema/mendam beneran".

### Voice range dinamis
Decode Opus → RMS dB → interpolasi whisper→normal→shout. `/av voicerange` + API
override. Shout dibatasi permission `astolfo.shout`.

### Noise cancellation (server-side, opt-in)
Noise gate PCM + Tier 2 bake + preset. RNNoise/Speex native = roadmap (belum bundle).

### Playback audio (`/astolfo play`)
- mp3 (JLayer), ogg/vorbis (JOrbis low-level public API, pure-Java, tanpa SPI),
  wav (javax.sound). Resample 48k Lanczos HIGH.
- Target: player, world, all, group, **location** (LocationSoundPacket per-listener).
- **Preset:** NONE PHONE RADIO MEGA CAVE KAWAII LOFI.
- **Pitch:** rate-based shift.
- Playlist persisten (playlist.yml).
- Folder audio auto-create `plugins/AstolfoVoice/audio/`.

### Visual & UX
- `AstolfoStyle` palet pink candy konsisten. Dekorasi imut hand-crafted (bukan
  emoji-spam generik). Bell feedback. Tab completion context-aware.

### API & integrasi
- `AstolfoApi` via ServicesManager. `playAtLocation` locational beneran.
- PlaceholderAPI expansion.
- PrivateChannel API (distance finite, audibleNearby option).
- Soft-hook Denizen/Skript via ServicesManager.
- Mute via MuteHolder (v0.2.3).

### Concurrency
Async penuh virtual thread (Java 25). UDP reader → bounded queue → virtual thread
per paket. Raytrace main-thread (Bukkit aman), cache per tick, DSP worker baca cache.
ThreadLocal Opus. TTL drop + bounded readByteArray. O(1) lookup address→UUID.

- **AstolfoOverride.java** — static holder voice range override per-player.
- **PrivateChannelRegistry.java** — static holder channel privat.

## 8. Proses Build & Release (loop yang dipakai)

1. Edit kode (prioritas compat-safe).
2. `.\gradlew.bat :AstolfoVoice:compileJava --offline --rerun-tasks` → verify
   BUILD SUCCESSFUL via script `_chk.ps1` (baca build_log.txt).
3. Scan TODO/stub/empty via `_scan*.ps1` (harus 0 marker).
4. Bump versi di root `build.gradle` (`allprojects { version = 'X.Y.Z' }`).
5. `.\gradlew.bat :AstolfoVoice:shadowJar --offline --rerun-tasks` → verify jar.
6. Bersihin temp script (`del _*.ps1 build_log.txt`).
7. `git add -A && git commit -F .git/COMMIT_MSG.txt && git tag -a vX.Y.Z -F ...`
8. `git push origin main && git push origin vX.Y.Z`

## 11. Catatan Teknis Penting (gotchas)

- **JOrbis OGG decode:** `VorbisFile.read(...)` package-private → gak bisa dari luar
  package. Pakai low-level public API: `SyncState`/`StreamState`/`Page`/`Packet`/
  `Info.synthesis_headerin`/`DspState.synthesis_*`/`Block.synthesis`. `Comment.clear()`
  juga package-private → jangan panggil (pakai `init()` saja). Output LE 16-bit
  interleaved → `bytesLeToMonoShorts`.
- **SVC-Bukkit pakai Concentus (Opus pure-Java)**, bukan native Opus → gak perlu
  bundle native Opus. Hanya RNNoise yang native (roadmap).
- **Channel namespace WAJIB `voicechat:*`** (bukan astolfo). Client SVC kirim ke situ.
- **FriendlyByteBuf** = thin wrapper, VarInt/UUID/UTF byte-exact netty. Self-contained
  (gak depend netty langsung, supaya format wire gak bergantung versi netty).
- **ServerVoiceEvents** perlu `setPlugin(this)` sebelum `init()` (Bukkit scheduler
  butuh plugin). Main onEnable sudah lakukan ini.
- **PrivateChannel distance:** pakai finite large (capped `maxPlayerRangeOverride`,
  default 1e6), BUKAN `Float.MAX_VALUE` (NaN/overflow di client).
- **AstolfoStyle** pakai Adventure `Component` (Paper 1.21+). `ChatColor` legacy
  gak dipakai lagi (kecuali fallback console). Component immutable → chain `.append()`.
- **`astolfo.png` (69x90) + `astolfo2.png` (238x183)** ada di root repo. Path di
  root docs: `astolfo.png`. Path di `AstolfoVoice/docs/`: `../../astolfo.png`.
  Path di `AstolfoVoice/README.md`: `../astolfo.png`.
- **Repo private** → raw URL image butuh token; fetch_url anonim 404 untuk halaman
  release. Verify GitHub via REST API pakai token GCM.
- **`.gitignore`** cover: `.gradle/`, `build/`, `_*.ps1`, `_*.txt`, `build_log.txt`,
  `dep_log.txt`, `*.log`, `svc-ref/`, `svc-verify/`, IDE files.
